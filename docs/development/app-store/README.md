## 🚀 快速使用

### 发布新版本流程
1. 构建发布包：`./scripts/build_release.sh`
2. 参考 `APP_STORE_INFO.md` 填写应用信息
3. 按照 `APP_STORE_PUBLISH_GUIDE.md` 进行提交
4. 使用 `APP_STORE_COPY.md` 中的文案内容

### 应用信息模板
所有必要的应用信息都在 `APP_STORE_INFO.md` 中预先准备好，包括：
- 简体中文和英文描述
- 版本更新说明模板
- 关键词推荐
- 分类和年龄分级建议
